# ⌨️ Keyboard Shortcuts

**Complete keyboard shortcut reference for power-user navigation**

---

## Overview

SaaSIQ supports keyboard shortcuts for fast navigation and common actions. Press **`?`** anywhere in the app to open the shortcuts overlay.

!!! tip
    On macOS use `⌘` (Command), on Windows/Linux use `Ctrl`.

---

## Global Shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `⌘ + K` / `Ctrl + K` | Open **Global Search** | Anywhere |
| `⌘ + /` / `Ctrl + /` | Open **AI Copilot** | Anywhere |
| `?` | Show **Keyboard Shortcuts** overlay | Anywhere (not in text fields) |
| `Escape` | Close current modal, panel, or dropdown | Anywhere |
| `⌘ + Shift + N` | **Quick action** — Create new item | Anywhere |

---

## Navigation Shortcuts

SaaSIQ uses **key chords** for navigation. Press `G`, release, then press the second key.

| Chord | Destination |
|-------|-------------|
| `G` then `D` | **Dashboard** (Overview) |
| `G` then `S` | **SaaS Discovery** |
| `G` then `I` | **Spend Intelligence** |
| `G` then `U` | **Usage Analytics** |
| `G` then `C` | **Compliance & Risk** |
| `G` then `T` | **Contracts** |
| `G` then `P` | **Policies** |
| `G` then `N` | **AI Insights** |
| `G` then `A` | **AI Copilot** |
| `G` then `O` | **Offboarding** |
| `G` then `R` | **Renewals** |
| `G` then `B` | **Benchmarks** |
| `G` then `E` | **Department Costs** |
| `G` then `L` | **Alerts** |
| `G` then `X` | **Settings** |

<details markdown>
<summary>💡 How Key Chords Work</summary>

A **key chord** is a two-step shortcut:

1. Press and release the first key (`G` for "Go to")
2. Within 1 second, press the second key (e.g., `D` for Dashboard)

If you press `G` and don't follow up within 1 second, the chord resets. Key chords are disabled when your cursor is in a text input field.

</details>

## Modal & Panel Shortcuts

| Shortcut | Action |
|----------|--------|
| `Escape` | Close the current modal or panel |
| `Enter` | Confirm / Submit the active modal |
| `Tab` | Move focus to the next field |
| `Shift + Tab` | Move focus to the previous field |
| `Space` | Toggle checkbox or radio button |

---

## Table Shortcuts

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Move selection up/down in a table |
| `Enter` | Open details for selected row |
| `⌘ + A` / `Ctrl + A` | Select all rows (when in a table) |
| `⌘ + Shift + E` | Export current table to CSV |

---

## AI Copilot Shortcuts

| Shortcut | Action |
|----------|--------|
| `⌘ + /` / `Ctrl + /` | Open Copilot chat panel |
| `Enter` | Send message |
| `Shift + Enter` | New line in message |
| `↑` | Recall previous message |
| `Escape` | Close Copilot panel |

---

## Customization

!!! note
    Keyboard shortcuts are not currently customizable. This feature is planned for a future release. You can request custom shortcuts by contacting support@saasiq.io.
